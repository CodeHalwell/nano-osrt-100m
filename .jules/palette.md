## 2024-05-18 - Chatbot Reasoning and Autofocus
**Learning:** Gradio's Chatbot component has built-in support for rendering `<think>` tags as separate, collapsible reasoning blocks via the `reasoning_tags=[("<think>", "</think>")]` and `type="messages"` arguments. Textboxes can also be set to `autofocus=True` to immediately capture user input without requiring a click.
**Action:** Use these features to improve the user experience of interacting with reasoning models.
