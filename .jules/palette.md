## 2025-02-27 - Gradio Parameter Tooltips
**Learning:** Users often struggle with advanced LLM generation parameters (temperature, top-p, etc.) when testing models. Gradio's `info` parameter on sliders provides an excellent, built-in way to add contextual tooltips without cluttering the UI with extra Markdown blocks.
**Action:** Always utilize the `info` parameter for complex configuration inputs in Gradio interfaces to provide immediate, inline documentation.
