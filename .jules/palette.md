## 2026-05-02 - Gradio UI Enhancements: Autofocus and Tooltips
**Learning:** Gradio UI components like `gr.Textbox` can immediately capture user focus using `autofocus=True`, reducing initial friction, while configuration inputs like `gr.Slider` can significantly improve discoverability with tooltips via the `info` property without cluttering the screen.
**Action:** Consistently apply `autofocus=True` to the primary input field on interactive web apps and add explanatory `info` tooltips to technical parameters to enhance usability and accessibility.
