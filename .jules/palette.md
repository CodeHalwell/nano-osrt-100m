## 2025-02-24 - Gradio Slider Tooltips
**Learning:** Gradio sliders and other input components support an `info` parameter natively, which provides helpful inline text directly beneath the label without needing custom tooltips or extra markdown blocks. This is excellent for explaining technical terms (like Top-p, Top-k).
**Action:** Use the `info` parameter on complex input components in Gradio to provide immediate context, reducing cognitive load without cluttering the UI with separate documentation.
