## 2025-05-11 - Gradio UI Autofocus and Tooltips
**Learning:** Gradio UI components like `Textbox` and `Slider` can be quickly improved using native properties like `autofocus=True` and `info="..."` respectively, which greatly improves UX and user guidance without needing complex custom CSS or external dependencies.
**Action:** Always leverage built-in properties such as `autofocus` for main inputs and `info` for technical sliders to guide users gracefully in Gradio applications.
