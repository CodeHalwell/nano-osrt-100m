## 2024-05-24 - Leverage Native Gradio Usability Features
**Learning:** Gradio UI components like `gr.Textbox` and `gr.Slider` have built-in parameters (`autofocus=True` and `info="tooltip text"`) that provide immediate accessibility and usability wins without needing custom CSS or complex component rewrites.
**Action:** When building or enhancing Gradio interfaces, always check for native attributes like `info` for tooltips and `autofocus` for primary inputs to guide users effortlessly.
